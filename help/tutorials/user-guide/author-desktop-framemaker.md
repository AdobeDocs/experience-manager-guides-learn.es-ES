# Trabajar con FrameMaker {#id221BH0L03XA}

AEM FrameMaker admite la creación y publicación del contenido en las guías de los usuarios de la. FrameMaker se envía con un conector incorporado a Adobe Experience Manager. FrameMaker también proporciona una interfaz fácil de usar que le permite mantener las versiones de sus documentos en un entorno distribuido y colaborativo. Puede crear el contenido en FrameMaker y publicarlo en distintos formatos: PDF, HTML 5, EPUB y DITA. También puede realizar las distintas operaciones de administración de archivos, como desprotección, desprotección con dependientes, protección, actualización, etc. Para obtener más información, consulte [Ayuda de FrameMaker](https://help.adobe.com/en_US/framemaker/using/index.html).
